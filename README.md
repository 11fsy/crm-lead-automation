# crm-lead-automation

## Overview

Designed an automated lead capture pipeline integrating Google Forms, Make, and Salesforce CRM. The system captures inbound form submissions and automatically creates structured Lead records in Salesforce.

## System Architecture

Google Form (Lead Capture)
        ↓
Make Automation
        ↓
Field Mapping & Validation
        ↓
Salesforce Lead Creation
        ↓
Salesforce Flow Automation

## Tools

- Salesforce CRM (Developer Edition)

- Make (Automation Platform)

- Google Forms

## Key Features

- Automated lead capture

- Field mapping between form responses and CRM schema

- Data validation

- Salesforce Flow automation
